# 07 — Cadence, Team & Workflow

## Weekly editorial workflow

```
   Editorial Inputs
         │
         ▼
   Weekly Editorial Meeting   →  "What is this week's Human Question?"
         │
         ▼
   Newsletter Published   (Mon)
         │
         ▼
   Video Essay Produced   (Tue)
         │
         ▼  (Wed–Sun) repurpose into:
   ├─ 📰 Newsroom episodes      (timely)
   ├─ 🧠 AI Explained episodes  (educational)
   ├─ 🎥 Essay clips            (thought leadership)
   ├─ X thread + daily X commentary
   ├─ Reddit discussions
   └─ LinkedIn post

   One idea. Many formats.
```
